1) DDL (Data Definition Language)

  CREATE

  ALTER 

  DROP

  TRUNCATE 

2) DML

	Insert : used to insert data into tables. 
Insert into table_name(field1,field2,field3...) values(value1,value2,value3...);

Update : used to update the table records. 

Update table_name set field1=value, field2=value, field3=value, field4=value {where condition}


Delete : Used to delete all or particular records from table. 

Delete from table_name {where condition}
