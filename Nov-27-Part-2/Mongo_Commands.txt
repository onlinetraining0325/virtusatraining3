✅ 2. Print dynamically for all fields (unknown structure)

If you don’t know the field names in advance:

for (Document d : col.find()) {
    for (String key : d.keySet()) {
        System.out.println(key + " : " + d.get(key));
    }
    System.out.println("-------------------");
}

✅ 3. Print as a formatted record
for (Document d : col.find()) {
    int id = d.getInteger("id");
    String name = d.getString("name");
    double salary = d.getDouble("salary");

    System.out.printf("ID: %d  Name: %s  Salary: %.2f%n", id, name, salary);
}
